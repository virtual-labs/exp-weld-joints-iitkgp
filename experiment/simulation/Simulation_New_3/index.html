Simulation

Instructions:

  • Before starting the process read the instructions completely
  • Select the button Types of Weld Joints or Weld Position Simulation
  • If Types of Weld Joints is selected follow the steps below
    • Select the type of Weld joint
    • Select the Groove type of weld as per the requirement.
    • Select the Weld position from (1G, 2G, 3G, 4G) where you want to perform the welding operation.
    • Observe the image and its position as per your selection
    • Click on the Stop button to stop the simulation
    • Repeat the experiment by selecting a different Weld joint, Groove type, and Weld position.
  • If Weld Position Simulation is selected follow the steps below
    • Select the Weld position from (1G, 2G, 3G, 4G) where you want to perform the welding operation.
    • Click on the Weld on button to start the simulation
    • Observe the welding simulation as per your selected position
    • Click on the Stop button to stop the simulation
    • Repeat the experiment by selecting a different Weld Position.
