Simulation

Instructions:

  1. Before starting the process read the instructions completely
  2. Press Start button to start the simulator.
  3. Select the type of Weld joint
  4. Select the Groove type of weld as per the requirement.
  5. Select the Weld position from (1G, 2G, 3G, 4G) where you want to perform the welding operation.
  6. Click on the Weld on button to start the simulation
  7. Click on the Stop button to stop the simulation
  8. Repeat the experiment by selecting a different Weld joint, Groove type, and Orientation.
