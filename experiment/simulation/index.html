Study and design of Weld's joints

Instructions:

  1. Before starting the process read the instructions completely
  2. Press Start button to start the simulator.
  3. Select the type of Weld joint
  4. Select the sub-type of weld as per the requirement.
  5. Select the weld position from (1G, 2G, 3G, 4G) where you want to perform the welding operation.
  6. Click on the Stop button to stop the simulation
